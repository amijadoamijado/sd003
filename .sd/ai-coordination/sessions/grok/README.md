# sessions/grok/

Grok が **Lead mode** で中〜長時間作業するときの任意メモ置き場。

## いつ書くか

- セッションが複数時間にまたがる
- 途中で Claude / Codex / agy に渡す可能性がある
- クラッシュ復帰用に進捗を残したい

## いつ書かなくてよいか

- 短い相談・Fast Review / Fast Implement
- 案件ID付きの正式報告がある（`workflow/review/` を使う）

## 書き方

- ファイル名例: `YYYYMMDD-HHmm-topic.md` または日付フォルダ
- 最小項目: 目的 / 触ったパス / 検証結果 / 残タスク / repo lock の所在
- 依頼書置き場ではない（依頼は `workflow/spec/{案件ID}/`）
