# 指示書ディレクトリ

## 命名規則
`YYYY-MM-DD-HH-MM-to-{宛先AI}.md`

例: `2025-12-21-11-30-to-gemini.md`

## テンプレート
```markdown
# 指示書: {タスク名}

**発行日時**: YYYY-MM-DD HH:MM
**発行元**: {自分のAI名}
**宛先**: {宛先AI名}
**優先度**: high | medium | low

## タスク概要
{タスクの説明}

## 具体的な要件
1. {要件1}
2. {要件2}

## 参照ファイル
- {参照すべきファイルパス}

## 品質基準
- {品質基準}

## 期限
{期限があれば記載}
```
